Periodic regulator reports

Generates and files a jurisdiction's periodic registers with the regulator SAFE on demand — no waiting for a period to close. Spain (DGOJ monitoring-system data model) files a daily detailed register (RUD) and a monthly totalized register (RUT); each register row totalises one player's settled betting activity for the period. Scenario: requirements/dgoj-periodic-reporting/ (repo root).

Generate & file registers

Each generation files ONE regulator batch to the SAFE (/safe/<MKT>/rud or /rut) covering every player active in the period — for Spain a single DGOJ <Lote> in the monitoring-system 3.3 format — and lands as XML under dataform-safe/<MKT>/. Re-filing the same period supersedes the previous receipt.
